# Reprograma Links

## Começando

É necessário ter `npm` ou `yarn` instalados em seu computador.

Instalar o VuePress

```
yarn global add vuepress

```

ou

```
npm install -g vuepress
```

## Executando o projeto

Para começar a contribuir é só executar o comando abaixo na raíz do projeto

```
vuepress dev docs
```

## Deploy

Para gerar os arquivos estáticos execute o comando abaixo na raíz do projeto
```
vuepress build docs
```
